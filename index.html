You can drag the blue objects and drop them to the grid.
It works better if you drag the top-left corners.

If you press 'r', the last dragged object will rotate.
RMB on an object - show the deletion menu. Then click 'delete!' - the object will be deleted.

In order to create an object, enter its name and dimensions, and choose where a road should be connected. Press 'Create!'.
Objects are created in one dedicated spot, so if you create multiple - don't miss them: they are under each other.
